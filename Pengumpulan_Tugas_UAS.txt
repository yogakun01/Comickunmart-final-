Mata Kuliah : <Pemrograman Mobile>
Judul       : ComicKunMart (Aplikasi Komik berbasis Expo)

Deskripsi Singkat:
- Aplikasi komik dengan peran Creator dan User.
- Baca chapter, kelola komik/chapter (Creator), beli/refund (User, status lokal).
- Mendukung deep linking: comickunmart:// untuk membuka halaman spesifik.

Link Video Demo:
- https://youtu.be/Qa979Ms4eyg

Source Code / GitHub Link:
- https://github.com/yogakun01/Comickunmart-final-.git

Cara Menjalankan (ringkas):
1) npm install
2) npx expo start

Catatan Teknis:
- Pembelian/refund tidak melakukan transaksi nyata; status disimpan lokal (AsyncStorage).
- Creator hanya melihat tombol Baca (tanpa Beli/Refund).
- Deep link aktif (contoh: comickunmart://comics/123).

Dokumentasi Tambahan:

- DEEP_LINKING_GUIDE.md & DEEP_LINK_README.md (panduan deep linking)
